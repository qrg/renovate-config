
# qrg/renovate-config

[Renovate shareable config presets](https://docs.renovatebot.com/config-presets/)

Enable Renovate in your repo and just extends in renovate.json.

## Default preset

```json
{
  "extends": ["github>qrg/renovate-config"]
}
```
